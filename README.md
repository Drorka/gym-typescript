# Fitness App

This responsive app provides information about fictional fitness classes, highlights the benefits of this fictional gym, and allows users to contact it through a form.

## Technologies Used

- React.js 
- TypeScript 
- TailwindCSS
- Framer Motion 
- Form Submit
- Vite 
- Cloudflare
